# ULibrary_api
Focus fullstack technical test (ULibrary) 
This is not deployed yet.  

### Tech Stack  
- Node js 
- Express
- jwt
- middlewares for routes security
- class validator
- PrismaORM
- Typescript
- Docker  
- Postgres
  
    
 ### Modules  
 - Users
 - Auth
 - Borrows
 - Books  
 - Roles  
   

### Instalation
 
 ```
 shell
 
 npm install  
 npm run dev  
 
 or with docker  
 docker build -t ulibrary .  
 docker-compose up
 
 ```
